# System Workflows

## Purpose

This document defines the major business workflows of the Employee Lifecycle and Workforce Management Portal.

These workflows serve as the foundation for architecture, database design, APIs, and implementation.

---

# Workflow 1: Recruitment

External Candidate
→ Application Submission
→ Screening
→ Interview
→ Offer
→ Hiring Decision

---

# Workflow 2: Onboarding

New Employee
→ Document Verification
→ Account Creation
→ Department Assignment
→ Manager Assignment
→ Orientation
→ Active Employee

---

# Workflow 3: Employee Management

Employee
→ Profile Management
→ Information Updates
→ Organizational Assignment
→ Workforce Tracking

---

# Workflow 4: Leave Management

Employee
→ Leave Request
→ Manager Review
→ Approval / Rejection
→ Leave Record Update

---

# Workflow 5: Performance Management

Employee
→ Goal Assignment
→ Performance Review
→ Feedback
→ Evaluation

---

# Workflow 6: Workforce Analytics

Employee Data
→ Aggregation
→ Reporting
→ Dashboard Insights
→ Leadership Review

---

# Workflow 7: Offboarding

Employee Separation Request
→ Approval
→ Asset Return
→ Access Revocation
→ Exit Process
→ Archived Employee Record

---

# Workflow 8: AI Knowledge Assistant

User Question
→ RAG Retrieval
→ Context Generation
→ AI Response
→ User Feedback
